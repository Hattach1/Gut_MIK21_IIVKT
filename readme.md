#  Steam Game Recommender

##  Описание проекта

Данный проект представляет собой веб-приложение с API, реализующее рекомендательную систему игр на основе данных Steam.

Система позволяет:

* искать игры
* выбирать любимые игры
* получать персонализированные рекомендации
* находить похожие игры

В основе лежит модель, использующая методы обработки текстов и вычисления схожести.

---

##  Используемая модель (ИИ)

В проекте реализована **контентная рекомендательная система** (Content-Based Filtering), основанная на:

* TF-IDF векторизации текстов
* cosine similarity
* анализе жанров и тегов
* дополнительном ранжировании по рейтингу и популярности

Модель учитывает:

* описание игры
* жанры
* категории (как теги)
* совпадения по ключевым игровым характеристикам

##  База данных

В проект включена готовая база `games.db`.

Это позволяет:

* не ждать загрузку данных из Steam
* сразу использовать рекомендации

При необходимости база может быть обновлена через API:

```
POST /admin/load-steam-games
POST /admin/update-game-reviews
```

#  User Manual

##  Назначение

Приложение позволяет получать рекомендации игр на основе ваших предпочтений.

---

##  Как пользоваться

###  Поиск игры

* Введите название игры в поле поиска
* Нажмите кнопку "Найти"
* К сожаоению загрузить подавлюящее большинсвто игр из steam в базу данных не представилось возможным из-за времени которое занимает их загрузка через steamAPI, поэтому в зараннее представленной в репозитории базе данных находятся только 347 игр.

---

###  Добавление в любимые

* В результатах нажмите кнопку "В любимые"
* Игра появится в списке выбранных

---

###  Получение рекомендаций

* Добавьте хотя бы одну игру
* Нажмите "Получить рекомендации"
* Система покажет список подходящих игр

---

###  Похожие игры

* У любой игры нажмите кнопку "Похожие"
* Отобразятся похожие игры

##  Модель рекомендаций

Используется content-based подход:

1. Формируется текстовое представление игры:

   * название
   * жанры
   * теги
   * описание

2. Применяется TF-IDF

3. Вычисляется cosine similarity

4. Добавляется ранжирование:

   * совпадение тегов
   * совпадение жанров
   * рейтинг
   * популярность

---

##  Формула ранжирования

Упрощённо:

```
score =
0.72 * similarity +
0.18 * tag_bonus +
0.08 * genre_bonus +
0.05 * rating +
0.02 * popularity
- penalty
```
